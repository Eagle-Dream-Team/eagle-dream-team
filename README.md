# 📘 Eagle University Portal 

## Overview

This project was developed as part of the **COMP1640 coursework**, where the team operated as a **cross-functional Agile Scrum unit**. Members assumed specialized technical roles while collaboratively supporting Scrum practices to ensure the *Eagle University Portal* met defined business, security, and quality requirements.

The project followed Agile principles, including iterative development, shared accountability, continuous testing, and collaborative documentation.
